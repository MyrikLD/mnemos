Use workspaces for projects — optional

                
By default memories are personal. For project-specific memory, create a workspace and give it a clear description — e.g. "Work memories for the Acme iOS project".

                
Before storing a memory Claude will check your workspaces and pick the right one automatically based on context. No need to mention the workspace name explicitly.

            

        


        


        
        

            
3

            

                
Share a workspace with teammates — optional

                
Open the workspace, click Generate invite link, and send it to your teammate. The link is one-time use and expires in 72 hours.

                
Once they join, everyone on the team references the same workspace name in their prompts and Claude shares memory across all members.
